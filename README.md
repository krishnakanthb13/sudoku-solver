# Sudoku Solver

A lightning-fast, modern Sudoku solver and validator featuring real-time conflict detection and an offline backtracking engine.

## üöÄ Features

*   **Instant Solver**: Uses a backtracking algorithm to solve 9x9 and 6x6 puzzles in milliseconds.
*   **Hint System**: Need a nudge? Get the correct value for one cell without revealing the entire solution.
*   **Random Generator**: Build random valid boards with a single click.
*   **Difficulty Modes**: Choose between **Easy, Medium, and Hard** modes for different challenge levels.
*   **Grid Selection**: Quickly toggle between **6x6** and **9x9** formats via a pop-up selection menu.
*   **Real-time Validation**: Detects conflicts (duplicate numbers in rows, cols, boxes) as you type.
*   **Dark Mode**: Fully supported dark/light themes that persist across sessions.
*   **Solved History Logs**: Automatically saves solved puzzles with precisely formatted timing ($\mu s$, ms, s). Restore puzzles anytime or delete specific entries (‚õî) from the history list.
*   **Log Export**: Save your Solved History Logs to a text file for record-keeping.
*   **Responsive Design**: Works perfectly on desktop and mobile.

## üì¶ Getting Started

### Prerequisites

*   Node.js (v16 or higher installed)

### Installation & Running

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/sudoku-solver.git
    cd sudoku-solver
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run the App**
    ```bash
    npm run dev
    ```
    *   **Windows Users**: You can simply double-click `run_app.bat` to install and start everything automatically.

4.  Open your browser to the URL shown in the terminal (usually `http://localhost:3000`).
